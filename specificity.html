What I Learned

The paragraphs with the span in it stays the original color because it doesnt have an id. The special span line has the special id so it turns blue. The line with myID turns red because I specified that the myID id is red. The same thing happens with myClass. For the second to last line the thing specified first gets precedence so that's why the second to last line is orange for the unique id and for the last line the last thing takes precedence so that is why it is yellow for the anotherclass class.
